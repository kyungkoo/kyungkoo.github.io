[iOS App Dev Tutorials] Getting Start With Scrumdinger

iOS App Dev Tutorials는 Scrumdinger 라는 앱을 만들면서 SwiftUI 를 배울 수 있는 학습 가이드다.

SwiftUI 살펴보기

SwiftUI 는 다음과 같은 특징을 갖는다.

  • Declarative syntax
  • Compositional API
  • Powerful layout system
  • Views reflect app data
  • Automatic accessibility support

Scrumdinger 살펴보기

Scrumdinger 는 크게 네 가지 화면으로 나눌 수 있다.

  • Scrum List
  • Scrum Detail and Edit
  • Metting Timer
